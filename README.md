## Installation

### Client

Client uses `vue-cli` to run a simple Vue app. 

```
cd client
npm install
npm run serve
```

### Server

Server uses `express.js`. Database is `JSON` based and uses `lowdb`.

```
cd server
npm install
npm run serve
```
